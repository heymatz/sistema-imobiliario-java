# Sistema Imobiliário

Sistema desenvolvido em Java para gerenciamento de imóveis, clientes e operações imobiliárias, criado como projeto de estudos com foco em Programação Orientada a Objetos e implementação de operações CRUD.

## Tecnologias

- Java
- Programação Orientada a Objetos (POO)
- CRUD

## Funcionalidades

- Cadastro de imóveis
- Cadastro de clientes
- Consulta de registros
- Atualização de informações
- Remoção de registros

## Objetivos de Aprendizagem

Durante este projeto foram praticados conceitos como:

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Organização de código
- Lógica de programação
